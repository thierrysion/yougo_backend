Questions fréquentes

Comment commander une course ?

1. Ouvrez l’application Yougo
2. Entrez votre lieu de départ et votre destination
3. Choisissez le type de véhicule souhaité
4. Confirmez la course et attendez la confirmation du chauffeur

J’ai oublié quelque chose dans le véhicule

Contactez-nous dans les 24 heures via l’application → « Aide » → « J’ai perdu un objet ».
Indiquez le numéro de course et décrivez l’objet. Nous contacterons le chauffeur.

Comment payer ma course ?

Vous pouvez payer par Mobile Money (Orange Money, MoMo) ou espèces (selon disponibilité dans votre ville).

Annulation d’une course

Vous pouvez annuler gratuitement dans les 2 premières minutes. Passé ce délai, des frais d’annulation peuvent s’appliquer.

Problème avec un chauffeur

Après la course, notez le chauffeur et ajoutez un commentaire. Pour un incident grave, contactez-nous immédiatement via « Aide » → « Signaler un problème ».
